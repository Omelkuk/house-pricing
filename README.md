 Структура проекта
Ниже представлено описание ключевых модулей и директорий проекта:

Основные компоненты (Core)
main.py: загрузка данных, запуск Optuna для тюнинга, финальное обучение и интеграция с MLflow

preprocessor.py: препроцессинг и фича инжиниринг

model_factory.py: чтобы вытаскивать удобно модельки и метрики

data_loader.py: Отвечает за безопасную загрузку CSV-файлов и проверку целостности данных перед началом обучения

logger.py: Кастомный логгер для красивого и структурированного вывода процесса в консоль. (я  не прикрепил журнал, так как mlflow решил задействовать)

config/config.yaml: конфиг :)


eda.ipynb: Jupyter Notebook с анализом данных, добавил фича импортанс


MLOps и Окружение
mlruns/: Локальное хранилище данных MLflow с историей всех запусков, метриками

requirements.txt: Список всех зависимостей, необходимых для воспроизведения среды обучения.

.gitignore: Инструкции для Git по исключению временных файлов, кэша и виртуального окружения из репозитория.
